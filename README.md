# Fusion

# An interpretted C like programming language


Hello world example:
```C
define main()
{
    var text = "Hello world!";
    printfn(text);
};
main();
```
To install the runtime for linux, download https://github.com/KickdeGans/Fusion/blob/main/install.sh.

How to install:
```C
sudo chmod +x ./install.sh
sudo ./install.sh
```
